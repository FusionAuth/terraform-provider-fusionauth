# FusionAuth Provider

This provider is used for setting up [FusionAuth](https://fusionauth.io)

## Argument Reference

* `api_key` - (Required) The API Key for the FusionAuth instance
* `host` - (Required) Host for FusionAuth instance

## Resources Available

* API Key
* application
* application/{application_id}/role
* email
* form
* form field
* group
* key
* imported key
* lambda
* identity provider

    - Google
    - Apple
    - External JWT
    - SAML v2
    - Sony PSN
    - Steam

    - Xbox
* themes
