# FusionAuth Provider

This provider is used for setting up [FusionAuth](https://fusionauth.io)

## Argument Reference

* `api_key` - (Required) The API Key for the FusionAuth instance
* `host` - (Required) Host for FusionAuth instance

## Resources Available

* application
* application/{application_id}/role
* email
* form
* form field
* group
* key
* imported key
* lambda
* indentity provider
    - OpenID Connect
    - Google
    - Apple
    - External JWT
* themes
* user
* user action
* webhook
* tenants
